# 📘 BPSC TRE Math Progress Tracker

A mobile-first, offline-ready **Progressive Web App (PWA)** designed for aspirants of the **BPSC Teacher Recruitment Exam (Math)**. Track your study progress across video lectures, NCERT chapters, and Previous Year Questions (PYQs).

---

## ✨ Features

- **Topic-wise Tracking:** Comprehensive list of all BPSC Math topics.
- **Three-way Progress:** Track Video Lectures, NCERT Reading, and PYQ Practice for each subtopic.
- **Dynamic Dashboard:** Visual progress ring, section readiness, and daily activity charts.
- **Offline First:** Works without internet after installation (PWA).
- **Data Persistence:** Automatically saves your progress to your browser's local storage.
- **Dark Mode UI:** Premium, distraction-free aesthetic.

---

## 🚀 How to Use (उपयोग कैसे करें)

### 1. Topics View (विषय ट्रैकर)
- **Topics List:** मुख्य स्क्रीन पर सभी विषयों की सूची दिखेगी।
- **Expand/Collapse:** किसी टॉपिक पर क्लिक करके उसके अंदर के सब-टॉपिक्स (Sub-topics) देखें।
- **Marking Progress:** 
    - 🎬 **Video:** वीडियो लेक्चर देखने के बाद क्लिक करें।
    - 📚 **NCERT:** महत्वपूर्ण रीडिंग पूरी होने पर क्लिक करें।
    - 📝 **PYQ:** पुराने प्रश्न हल करने के बाद क्लिक करें।
- **Auto-Sync:** आपकी प्रोग्रेस अपने आप सेव होती रहती है।

### 2. Dashboard (डैशबोर्ड)
- **Overall Readiness:** अपनी कुल तैयारी का प्रतिशत देखें।
- **Section Stats:** अलग से देखें कि आपकी NCERT या PYQ की तैयारी कितनी हुई है।
- **Activity Chart:** पिछले 14 दिनों की अपनी पढ़ाई की फ्रीक्वेंसी देखें।

### 3. Installation (इंस्टॉल कैसे करें)
- **Mobile:** ब्राउज़र में खोलें और "Add to Home Screen" बैनर पर क्लिक करें।
- **Desktop:** एड्रेस बार में दिखने वाले 'Install' आइकन पर क्लिक करें।

## 🛠️ Tech Stack
- **Frontend:** HTML5, Vanilla CSS3 (Custom Variables)
- **Logic:** Vanilla JavaScript (ES6+)
- **Storage:** Web Storage API (LocalStorage)
- **PWA:** Service Workers, Web App Manifest
